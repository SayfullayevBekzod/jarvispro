# Jarvis - O'zbekcha Voice Assistant

O'zbek tilida gaplashadigan va tushunadigan sun'iy intellekt yordamchisi.

## O'rnatish

```bash
pip install -r requirements.txt
```

## Ishga tushirish

```bash
python main.py
```

## Sozlash

`config.py` faylida Muxlisa API kalitini kiriting:
```python
MUXLISA_API_KEY = "sizning_api_kalitingiz"
```

## Buyruqlar

- "Salom Jarvis" - Salomlashish
- "Soat necha?" - Vaqtni aytish
- "Chrome ochib ber" - Dasturni ochish
- "5 plyus 5" - Hisoblash
- Va boshqa ko'plab buyruqlar...
